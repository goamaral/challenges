# Challenge
## Description
In this challenge I was asked to build a simple users CRUD api using GRPC and Golang.
I should also notify user events (create, update, delete).

## Features
- [X] Health check
- [X] Create user
- [X] Update user
- [X] Delete user
- [X] List users
- [ ] Get user
- [X] Publish user events (create, update, delete)
